# IDA_Plugin_IID_to_String
A plugin for IDA that converts IID/GUID data structures to string and adds comments where the IID is referenced

## Installation
Copy "iid-to-string.py" to your IDA Pro's "plugins" directory

## Usage
In the disassembly window, right click an IID/GUID data structure, and select "IID to String" in the context menu
